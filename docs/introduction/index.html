Online psychological experiments and surveys

This documentation site refers to the ILCB server, how it was built and is run, and refers also to several other external resources that one can use for experimenting online.

#ILCB

The Institute for Language, Communication and the Brain is a French joint research institute bringing together experts in linguistics, neuroscience, psychology, medicine and computer science to understand and to model the way that language functions. The main objective of this 10-year scientific adventure is to create a generic model of the processing of language and its cerebral bases. The institutional support comes from CNRS, INSERM, Aix-Marseille Université and Université d'Avignon.

#Resources

#Build, manage and promote an online experiment

#Cubius

Cubius is an Harvard University based solution for dynamically building static web sites to be hosted on Amazon servers.

To top

#JsPsych

JsPsych is a JavaScript library for running behavioral experiments in a web browser. The library provides a flexible framework for building a wide range of laboratory-like experiments that can be run online.

To top

#Lab.js

Lab.js is a free, open, online study builder for the behavioral and cognitive sciences.

To top

#OSWeb

OSWeb is an online runtime for OpenSesame experiments.

To top

#PsychopyJs

PsychopyJs added the facility to run studies online using a high-performance, hardware-accelerated, port of the PsychoPy Python library.

To top

#JATOS

JATOS, Just Another Tool for Online Studies, helps you set up and run your online studies on your own server.

To top

#Pavlovia

Pavlovia, where behaviour is studied. Supported by the Wellcome Trust and the University of Nottingham.

To top

#The experiment factory

The experiment factory builds Docker containers ready to be deployed on servers

To top

#Netlify

Netlify hosts studies and take care of data collection. Their service is easy to use, free for basic studies, and inexpensive for large ones.

To top

#Openlab

Openlab](https://www.open-lab.online) takes care of running experiments and collecting, storing and managing data. The basic version of the application is available for free.

To top

#SoSci

SoSci is a solution for online questionnaires

To top

#Sona systems

Sona systems virtualizes every aspect of the sign-up process, eliminating the frustration of paper-based methods and bulletin boards.

To top

#Prolific

Prolific collects high quality responses from people around the world within minutes.

To top

#Mechanical Turk

Mechanical Turkis a crowdsourced market place

To top

#Labvanced

Labvanced is probably a cloud solution for professional web based experiments

To top

#Limesurvey

Limesurvey is an open source survey software

To top

#Qualtrics

Qualtrics is an online survey platform

#ILCB server

We opted for a cloud web server hosted by DigitalOcean with a JATOS experiment manager. Regular PHP/MySQL/JavaScript experiments can also be run on this server.
